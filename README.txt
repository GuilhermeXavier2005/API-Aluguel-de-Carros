------> Introdução:

		API de Aluguel De Carros

	O principal objetivo do projeto é mesclar habilidades de programação e a solução de problemas reais, aplicar lógica e otimização de resultados para estabelecer fluidez entre as informações. O usuário é capaz de fazer um cadastramento com suas informações, consultar, alugar e ter acesso aos seus dados bem como os carros alugados.
	Acessar como administrador permite que haja maior controle do fluxo de informações dentre os usuários e os carros contidos, permitindo maior variedades de opções como, adicionar, excluir, modificar a ficha de disponibilidade de carros, contato com os clientes e fornecedores.
